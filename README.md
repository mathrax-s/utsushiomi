たくさんの小さな木のオブジェがのったテーブルがつながっています。その木のオブジェは触れると音がします。
音を頼りにしてもいいし、手触りを頼りにいいし、実は光や香りも変化していて、いろいろな感覚を頼りにしながら、前に歩いていく体験型の作品です。
この作品は、目が不自由でも、耳が不自由でも、車椅子の高さでも、そして小さな子供の目線でも、それぞれの感覚で体験することができるように作りました。
